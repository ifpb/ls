## Vite

[Vite doc](https://vitejs.dev/guide/):

```bash
$ npm create vite@latest invest-app -- --template vanilla

Need to install the following packages:
  create-vite@4.4.1
Ok to proceed? (y)

Scaffolding project in /home/.../invest-app...

Done. Now run:

  cd invest-app
  npm install
  npm run dev
```

```text title="Arquivos"
invest-app
├── counter.js
├── index.html
├── javascript.svg
├── main.js
├── package.json
├── public
│   └── vite.svg
└── style.css
```

```bash
$ cd invest-app
$ npm install
$ npm run dev

> invest-app@0.0.0 dev
> vite

```
